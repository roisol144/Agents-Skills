# Install: commit-message

**Scope:** User — install once, available across all your projects.

---

## What this skill does

Analyzes your staged changes, reads the project's commit history to match its style, and writes a clean conventional commit message ready to commit.

**Invoke with:** `/commit-message`

---

## Prerequisites

- `git`
