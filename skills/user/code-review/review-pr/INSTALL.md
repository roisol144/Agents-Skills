# Install: review-pr

**Scope:** User — install once, available across all your projects.

---

## What this skill does

Performs a structured code review of any open pull request — checking correctness, security, test coverage, performance, and style consistency. Produces clear, prioritized comments (blocker / suggestion / nit) with a final verdict.

**Invoke with:** `/review-pr`

---

## Prerequisites

- `gh` (GitHub CLI) — [install guide](https://cli.github.com/) then run `gh auth login`
